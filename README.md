# 💰 CareCost AI – Healthcare Payment Forecasting Model

## 📌 Overview

CareCost AI is a machine learning-based project designed to **predict healthcare payment costs** using provider service details and patient risk characteristics.
The goal is to help healthcare providers and analysts **identify cost patterns and reduce revenue leakage**.

---

## 🚀 Features

* 📊 Data Cleaning & Preprocessing
* 📈 Exploratory Data Analysis (EDA)
* 🤖 Machine Learning Models for Cost Prediction
* 📉 Model Evaluation (R² Score, MAE, RMSE)
* 🌐 (Optional) Streamlit Web App for Predictions

---

## 🧠 Tech Stack

* Python
* Pandas, NumPy

* Streamlit

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/CareCost-AI-Healthcare-Payment-Forecasting-Model.git
cd CareCost-AI-Healthcare-Payment-Forecasting-Model
```

### 2. Create virtual environment (recommended)

```
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

### 3. Install dependencies

```
pip install -r requirements.txt
```

---

## ▶️ Run the Project

### Run Python Script

```
python your_script.py
```

### Run Streamlit App

```
streamlit run app.py
```
